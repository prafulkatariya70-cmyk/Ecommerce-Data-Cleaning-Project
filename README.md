# 📊 E-commerce Data Cleaning & Analysis Project

## 🚀 Project Overview

This project focuses on cleaning and analyzing a messy e-commerce dataset containing over 20,000 rows. The dataset included multiple real-world issues such as missing values, duplicate records, inconsistent text formats, and mixed date formats.

The goal of this project was to transform raw, unstructured data into a clean and analysis-ready dataset using both Excel and Python.

---

##  Objectives

* Clean and preprocess messy real-world data
* Handle missing values and duplicates
* Standardize inconsistent data formats
* Perform data analysis and generate insights
* Demonstrate both manual (Excel) and automated (Python) data cleaning techniques

---

##  Tools & Technologies Used

* **Microsoft Excel** – Data cleaning and visualization
* **Python (Pandas)** – Data preprocessing and automation

---

## Data Cleaning Steps

* Removed duplicate rows to ensure data accuracy
* Handled missing values using appropriate strategies (mean imputation and placeholders)
* Standardized categorical columns (Country, Category, Product)
* Converted inconsistent date formats into a uniform structure
* Identified and handled outliers in numerical data
* Ensured data consistency and integrity

---

##  Analysis & Insights

* Revenue distribution by country
* Identification of top-performing products
* Category-wise revenue contribution
* Basic trend analysis using date data

---

## 📁 Project Structure

* **dataset/** → Raw and cleaned Excel datasets
* **scripts/** → Python scripts and datasets
* **dashboard/** → Visualizations and charts

---

##  Key Learnings

* Real-world data is often messy and requires structured cleaning
* Importance of handling missing values and inconsistencies
* Combining Excel and Python improves efficiency
* Data validation is crucial before analysis

---

## 📌 Conclusion

This project demonstrates strong data cleaning and preprocessing skills, which are essential for data analyst roles. It highlights the ability to work with messy datasets and extract meaningful insights using industry-relevant tools.
